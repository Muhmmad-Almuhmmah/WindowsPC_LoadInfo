# Windows PC LoadInfo
simple code to load software and hardware info About Windows OS ,BIOS and Board
This code reads the following information from the Registry
- Board Manufacturer
- Board Product
- Board Version
- System SKU
- BIOS Release Date
- BIOS Vendor
- BIOS Version
- System Family 
- System Manufacturer
- System ProductName
- Identifier
- Processor Name String
- Vendor Identifier
- Current Build
- Composition Edition ID
- Edition ID
- Current Version
- Current Type
- Product Name
- Release Id
- Display Version
- Registered Owner
